# TowerFlow Phase 1 Assumptions

## Purpose

This prototype is an engineering visualisation proof of concept. It demonstrates the static data chain from offline structural analysis to a mobile-friendly 3D tower viewer.

## Model Boundary

- Asset type: simplified triangular communication tower.
- Structural idealisation: 3D pin-jointed truss.
- Scope: steel tower superstructure only.
- Supports: three base nodes restrained in global X, Y, and Z translation.
- Loading: one simplified horizontal wind load case in the global X direction.
- Result focus: member axial force and support reaction visualisation.

## Exclusions

- No AS/NZS 1170.2 wind procedure.
- No AS 4100 member capacity check.
- No connection design.
- No foundation, anchor bolt, or base plate design.
- No certification wording.
- No reliability or probability assessment.

## Product Language Rule

All product-facing interface text, example data, reports, charts, schemas, and exported material must be in English.
